# Portfolio Santos

Site de portfólio pessoal desenvolvido por mim para apresentar meu perfil profissional, competências, formação e evolução na área de tecnologia.

---

## 📌 Sobre o projeto

Este é um site estático criado com HTML5, CSS e JavaScript, com foco em:

* Design moderno e profissional
* Responsividade total (desktop e celular)
* Organização clara de cursos por instituição
* Sinalização visual de status (Cursando, Concluído, Trancado)
* Animações suaves e experiência visual agradável

O objetivo é ter um portfólio simples, elegante e fácil de hospedar.

---

## 🚀 Tecnologias e bibliotecas utilizadas

* HTML5
* CSS3
* JavaScript
* Tailwind (via CDN)
* GSAP (animações)
* ScrollReveal
* AOS (animações on scroll)
* Lottie Player (animações leves)
* Splide (carousel)
* Font Awesome (ícones)

## 💻 Como executar localmente

### Método simples

Basta abrir o arquivo `portfolio_santos.html` diretamente no navegador.

### Método recomendado (servidor local)

Se preferir rodar em servidor local:

```bash
python -m http.server 8000
```

Depois acesse:

```
http://localhost:8000/portfolio_santos.html
```

---

## 🎨 Personalização

Você pode alterar facilmente:

* **Foto de perfil:** substituindo `assets/profile.jpg`
* **Email e contatos:** editando diretamente no HTML
* **Nome exibido:** alterando no título e na seção principal
* **Cursos e status:** modificando os textos e as classes de badge
* **Cores do site:** ajustando as variáveis de cor no CSS

---

## 🌐 Deploy

O site pode ser hospedado facilmente em:

### GitHub Pages

1. Criar repositório
2. Enviar os arquivos
3. Ativar GitHub Pages nas configurações

### Netlify

Basta arrastar a pasta do projeto para a plataforma e fazer o deploy.
