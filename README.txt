Site estático — Santos
---------------------

Arquivos:
- index.html : página principal
- static/login_padrao.png : imagem de avatar (placeholder)
- static/verificado.jpeg : selo de verificação (placeholder)

Uso:
1. Descompacte o arquivo site_santos.zip
2. Abra index.html em um navegador para ver o site estático
3. Para hospedar: envie a pasta para um host de arquivos estáticos (Netlify, Vercel, GitHub Pages, etc.)
4. Para adicionar fotos reais, substitua os arquivos em /static e atualize o conteúdo conforme necessário.

Observações:
- O site foi preparado para ser 100% funcional como site estático (sem backend).
- O botão "Enviar email" abre o cliente de email padrão via mailto:.
